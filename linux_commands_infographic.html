üí° Common Command Combinations

Find Large Files
find /home -type f -size +100M
Find files larger than 100MB in /home directory
Monitor Log Files
tail -f /var/log/syslog | grep error
Follow system log and filter for errors
Archive and Compress
tar -czf backup.tar.gz /home/user
Create compressed archive of directory
Extract Archive
tar -xzf backup.tar.gz
Extract compressed tar archive
Count Files in Directory
ls -1 | wc -l
Count number of files in current directory
Find Recently Modified Files
find . -mtime -7 -type f
Find files modified in last 7 days
Check Disk Usage by Directory
du -h --max-depth=1 | sort -hr
Show directory sizes sorted by largest
Find and Delete Empty Files
find . -type f -empty -delete
Find and remove all empty files
Show Top CPU Processes
ps aux --sort=-%cpu | head
Display processes using most CPU
Find Files by Pattern and Execute
find . -name "*.log" -exec rm {} \;
Find all .log files and delete them
Monitor Network Traffic
netstat -i 1
Show network interface statistics every second
Create Backup with Date
cp file.txt file_$(date +%Y%m%d).txt
Create backup file with current date
Show Listening Ports
netstat -tlnp | grep LISTEN
Display all listening network ports

üí° Pro Tips for Linux Command Line

Use Tab Completion
Press Tab to auto-complete commands, filenames, and paths. Double-tap to see all options.
History with !!
Use !! to repeat the last command, or !n to repeat command number n from history.
Pipe Commands Together
Use | to chain commands: ps aux | grep firefox | wc -l counts Firefox processes.
Use Wildcards
* matches anything, ? matches single character: rm *.tmp deletes all .tmp files.
Background Tasks with &
Add & at end of command to run in background: long_script.sh & continues in background.
Redirect Output
Use > to save output to file, >> to append: command > output.txt saves result.
Use alias for Common Commands
Create shortcuts: alias ll='ls -la' makes ll shorthand for detailed listing.
Check Command Manual
Use man command_name to read detailed manual pages for any command.
Use which and type
Find command locations with which, or use type to see if it's built-in, alias, or file.
Quick File Search
Use locate for fast file search, but run updatedb first to update the database.
Monitor in Real-time
Use watch command to run commands repeatedly: watch -n 2 df -h shows disk usage every 2 seconds.
Safe File Operations
Use -i flag for interactive mode: rm -i asks before deleting, cp -i before overwriting.

OSI Model

The Open Systems Interconnection model is a conceptual framework that standardizes network communication into seven distinct layers. Click on each layer to explore its function, protocols, and real-world examples.

7
Application Layer
+
Network services directly accessible to end-users
Function
Provides network services directly to applications and end-users. Handles user authentication, data formatting, and application-specific protocols.
Common Protocols
HTTP/HTTPS FTP SMTP DNS SNMP Telnet
Examples
Web browsers, email clients, file transfer programs, remote login applications
6
Presentation Layer
+
Data translation, encryption, and compression
Function
Transforms data between application and network formats. Handles encryption/decryption, compression/decompression, and character encoding.
Common Protocols
SSL/TLS JPEG GIF ASCII EBCDIC
Examples
Data encryption, image compression, character set conversion, data formatting
5
Session Layer
+
Establishes, manages, and terminates connections
Function
Manages communication sessions between applications. Controls dialogues and connections, handles session checkpoints and recovery.
Common Protocols
NetBIOS RPC SQL NFS SMB
Examples
Session establishment, authentication, session recovery, dialogue control
4
Transport Layer
+
Reliable data delivery and error recovery
Function
Provides reliable data transfer between end systems. Handles error detection, recovery, flow control, and segmentation.
Common Protocols
TCP UDP SPX SCTP
Examples
Port addressing, data segmentation, error correction, flow control
3
Network Layer
+
Routing and logical addressing
Function
Routes data packets between different networks. Handles logical addressing, path determination, and packet forwarding.
Common Protocols
IP ICMP OSPF RIP BGP
Examples
IP addressing, routing tables, packet forwarding, routers
2
Data Link Layer
+
Node-to-node data transfer and error detection
Function
Provides node-to-node data transfer and error detection/correction. Handles MAC addressing and frame synchronization.
Common Protocols
Ethernet Wi-Fi PPP ATM Frame Relay
Examples
MAC addresses, switches, bridges, network interface cards
1
Physical Layer
+
Physical transmission of raw data bits
Function
Transmits raw bit streams over physical medium. Defines electrical, mechanical, and procedural specifications.
Components
Cables Hubs Repeaters Modems Radio Waves
Examples
Fiber optic cables, copper wires, radio frequencies, voltage levels
